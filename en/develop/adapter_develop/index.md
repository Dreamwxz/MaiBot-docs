# MaiBot Development Guide

## MaiBot External Adapter Development
This section is for developers who want to enable communication between MaiBot and external software.
Within the existing MaiBot framework, your plugins should communicate with MaiBot using the standard MaiBot communication library [maim_message](https://github.com/MaiM-with-u/maim_message).
